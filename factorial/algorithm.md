#### Написать программу для вычисления факториала. Сделать вариант с рекурсией и без.

**Факториал** — это произведение всех натуральных чисел от `1` до `N` включительно.  

Например:  
- `1! = 1`  
- `2! = 1 × 2 = 2`  
- `3! = 1 × 2 × 3 = 6`  
- `N! = 1 × 2 × 3 × ... × N`  

Базовые случаи  
```text
Если N == 0 или N == 1:  
    вернуть 1  
Все если N == 0 или N == 1  

Если N > 1:  
    вернуть N × Факториал(N - 1)  
Все если N > 1
```

---

Рекурсия
```text
Вычислить факториал(N):  
    если N == 0 или N == 1:  
        вернуть 1  
    иначе:  
        вернуть N × Факториал(N - 1)  
```

---  

Без рекурсии
``` text
Вычислить факториал(N):  
    результат = 1  
    цикл для i от 2 до N:  
        результат = результат × i  
    конец цикл  
    вернуть результат  
```
