#### Дана квадратная матрица. Вывести ту часть, что находится ниже главной диагонали.

##### Алгоритм
1. Начало программы.
2. Объявить квадратную матрицу `matrix` заданного размера.
3. Проверить, является ли матрица квадратной:
    - То есть количество строк равно количеству столбцов.
4. Организовать двойной цикл:
    - Внешний цикл по строкам: `i от 0 до matrix.length - 1`
    - Внутренний цикл по столбцам: `j от 0 до i - 1`
        - Таким образом, мы проходим только по тем позициям, где `i > j`
5. На каждой итерации вывести или сохранить значение `matrix[i][j]`.
6. После обработки всех элементов завершить программу.
