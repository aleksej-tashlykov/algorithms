#### Создать программу, позволяющую вводить с клавиатуры число, сохранять его в целочисленной переменной и затем выводить переменную на консоль. Надо учесть, что пользователь может ввести любую строку. Если среди значащих символов есть только цифры, + или -, то строка должна быть преобразована в число.

**Задача** — ввести строку с клавиатуры и преобразовать её в целое число, если она представляет собой корректную запись целого числа (возможно, с `+` или `-` в начале).  
Игнорировать пробелы в начале и конце.

##### Примеры:
- Ввод: `"  123  "` → результат: `123`
- Ввод: `"-456"` → результат: `-456`
- Ввод: `"+789"` → результат: `789`
- Ввод: `"abc"` → ошибка → завершить программу
- Ввод: `"12a3"` → ошибка → завершить программу
- Ввод: `"  -  12"` → ошибка (пробел между `-` и `1`) → завершить

### Базовые случаи
- Если строка пустая
  завершить программу

- Если строка пустая после удаления пробелов:  
  завершить программу

- Если первый символ `+` или `-` есть, но за ним нет цифр:  
  завершить программу

- Если есть символы, не являющиеся цифрами, за пределами первого символа:  
  завершить программу

### Алгоритм

**Парсинг целого числа(строка):**  
 i = 0
 длина = длина строки

 // Пропуск пробелов в начале
 цикл пока i < длина и строка[i] == ' ':
  i = i + 1
 конец цикла

 если i == длина:
  завершить программу  // только пробелы

 знак = 1
 если строка[i] == '+':
  i = i + 1
 иначе если строка[i] == '-':
  знак = -1
  i = i + 1
 конец условия

 если i == длина:
  завершить программу  // был знак, но нет цифр

 число = 0
 цифра = 0

 цикл пока i < длина:
  код = код символа строки[i]

  если код >= 48 и код <= 57:  // '0' = 48, '9' = 57
   цифра = код - 48
   число = число * 10 + цифра
  иначе если строка[i] == ' ':  // пробел внутри — ошибка
   завершить программу
  иначе:
   завершить программу
  i = i + 1
 конец цикла

 если знак == -1:
  число = -число

 вывести число
