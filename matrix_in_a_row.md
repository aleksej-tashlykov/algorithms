#### Дана матрица. Вывести все элементы в одну строку

##### Алгоритм
1. Начало программы.
2. Объявить матрицу `matrix` с данными.
3. Создать пустую строку `result = ""` (или массив, если удобнее).
4. Организовать двойной цикл:
    - Внешний цикл перебирает строки (`i от 0 до matrix.length - 1`)
    - Внутренний цикл перебирает столбцы каждой строки (`j от 0 до matrix[i].length - 1`)
    - На каждой итерации добавлять текущий элемент `matrix[i][j]` к строке `result`, добавляя разделитель (например, пробел)
5. После обработки всех элементов вывести готовую строку.
6. Конец программы.
